Диаграмма C4 Container в drawio: [С4 Container](./С4_Container.drawio)
Требования к безопасности:
 - взаимодействие с партнером по TLS
 - двунаправленный ssl сертификат для проверки подлинности
 - обеспечить ротацию ssl сертификата в оговоренный с ИБ период
 - на стороне партнера должен быть white list разрешенных ip адресов системы и на стороне системы также должен быть white list разрешенных ip адресов партнера
 - необходимо защитить команды датчикам контрольной суммой HMAC
 - необходимо добавить аудит на команды управления умными устройствами
 - отправлять партнеру минимально необходимы данные, в случае необходимости отправить персональные данные - согласовать это со специалистом ИБ
 - данные биометрии хранить в системе в обезличенном виде
